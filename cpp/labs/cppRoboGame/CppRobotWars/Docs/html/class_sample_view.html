Подробное описание

Пример вида для битвы роботов.

Данный класс реализует обработку событий от модели (класс Kernel) Для получения возможности обработки события от ядра необходимо реализовать шаблонный интерфейс IObserver<T>, где T класс унаследованный от класса Event и содержащий параметры, связанные с событием
